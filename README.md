# 项目简介

## 核心功能
- 前后端代码及SQL文件
- 自动发货功能
- 用户在线咨询功能
- 首页水果推荐协同算法（按用户购买推荐）

## 适用场景
- 水果零售商
- 水果批发商
- 水果电商平台

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
